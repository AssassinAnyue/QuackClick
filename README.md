# 🦆 鴨子點擊遊戲 - Duck Clicker

一個 Cookie Clicker 風格的放置點擊遊戲，主題為可愛的鴨子養成與像素藝術風格。

## 🎮 遊戲概述

玩家通過點擊鴨子獲得 Quack 值，購買道具提升效率，並升級鴨子階段。遊戲目標是將鴨子從鴨蛋養成至絕對鴨，同時避免鴨力值耗盡導致遊戲結束。

## 🚀 快速開始

### 安裝依賴

```bash
npm install
```

### 運行開發伺服器

```bash
npm run dev
```

遊戲將在 `http://localhost:5173/` 啟動。

### 構建生產版本

```bash
npm run build
```

## 🎯 遊戲機制

### 核心數值系統

- **Quack 值**：遊戲主要貨幣，無上限，顯示在畫面最上方
- **鴨力值**：初始 0 點，每秒自動增加 1 點，達到上限時遊戲結束
- **點擊獲得**：基礎點擊 +1 Quack，可透過道具提升
- **自動生產**：每秒自動獲得 Quack，可透過道具提升

### 鴨子階段系統

鴨子共有 15 個成長階段，每個階段需升級 5 次才能進入下一階段。每升級一個階段，最大鴨力值減少 10，最低 30 點。

**階段列表**：
1. 鴨蛋 → 2. 黃鴨 → 3. 白鴨 → 4. 成年鴨 → 5. 至聖先鴨
6. 天啟鴨 → 7. 星界鴨 → 8. 混沌鴨 → 9. 永恆鴨 → 10. 超鴨神體
11. 鴨界意志 → 12. 原初之鴨 → 13. 鴨神皇 → 14. 多元鴨體 → 15. 絕對鴨

達到「絕對鴨」階段即勝利。

### 道具商店系統

左側顯示 20 種可升級道具，每個道具可升級多次。道具根據鴨子階段解鎖，提供各種增益效果：

- **點擊系**：提升點擊收益、連擊加成、點擊速度
- **自動系**：提升自動生產、生產效率、加速器
- **加成系**：全局倍率、專注加成、信仰池
- **特殊系**：彩羽共鳴、神聖共鳴、時間凍結、維度之門

**道具解鎖節奏**：
- 鴨蛋階段：解鎖鴨點擊器、鴨自動機
- 黃鴨階段：解鎖超級鴨點擊器、鴨式連擊模組
- 白鴨階段：解鎖鴨之風暴、鴨時間加速器
- 成年鴨階段：解鎖鴨燃引擎、閃電鴨核
- 至聖先鴨階段：解鎖鴨之專注、鴨之祝福
- ... 依此類推

### 遊戲平衡

- **早期遊戲**：成本降低 70%，確保玩家在前 2 分鐘能快速升級
- **增長曲線**：保持指數增長，確保中後期深度
- **解鎖機制**：道具根據階段逐步解鎖，避免信息過載

## 📁 項目結構

```
src/
├── components/          # React 組件
│   ├── Header.tsx       # 頂部信息欄
│   ├── DuckArea.tsx     # 鴨子點擊區域
│   ├── ItemShop.tsx     # 道具商店
│   └── GameOverModal.tsx # 遊戲結束模態框
├── types.ts             # TypeScript 類型定義
├── gameData.ts          # 遊戲數據（升級費用、階段列表）
├── upgradeItems.ts      # 道具升級數據
├── gameReducer.ts       # 遊戲狀態管理邏輯
├── utils.ts             # 工具函數（數字格式化、持久化）
├── App.tsx              # 主應用組件
├── main.tsx             # 應用入口
└── index.css            # 全局樣式
```

## 📝 開發說明

- 遊戲循環以 60 FPS 運行，每秒執行一次 TICK 操作來更新自動 Quack 生產和鴨力值
- 狀態管理使用 `useReducer` 集中處理所有遊戲邏輯，確保狀態更新的可預測性
- 道具效果計算使用緩存機制，提升性能
- 數字格式化使用千分位分隔符，不使用 K/M/B 簡寫
